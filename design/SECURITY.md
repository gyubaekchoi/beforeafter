# Security policy

## Supported versions

The versions of the project that are currently supported with security updates.

| Montage version | Release    | Supported                       |
| --------------: | :--------- | :------------------------------ |
|          ^3.0.0 | 2025-11-13 | :white_check_mark: Stable major |
|          ^2.0.0 | 2025-04-30 | :white_check_mark:              |
|          ^1.0.0 | 2024-05-07 | :x:                             |
|         <=1.0.0 | 2024-03-15 | :x:                             |

## Reporting a vulnerability

If you discover a security vulnerability, please report it through [GitHub Security Advisories](https://github.com/wanteddev/montage-web/security/advisories/new). **Do not open a public issue.**

### What to include

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact
