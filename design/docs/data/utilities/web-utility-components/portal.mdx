## Introduce

`React.createPortal` 을 컴포넌트로 사용할 수 있습니다.

```tsx
import { Portal } from '@wanteddev/wds';

const Demo = () => {
  return (
    <Portal>
      <div />
    </Portal>
  )
}

export default Demo;
```
