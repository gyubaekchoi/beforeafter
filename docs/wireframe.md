# 장기 사용 (B&A) 리뷰 - 상세 화면 와이어프레임

이 문서는 업데이트된 기획안(PRD)을 바탕으로 구성된 B&A 리뷰 기능의 텍스트 기반 상세 와이어프레임입니다. 개발 및 디자인 작업 시 요구사항을 직관적으로 확인할 수 있도록 화면 단위로 구성되었습니다.

---

## 1. [어드민] B&A 리뷰 설정 페이지
**설명:** 쇼핑몰 관리자가 B&A 리뷰 기능을 활성화하고 옵션을 설정하는 화면입니다.

```text
[ 헤더: 리뷰 설정 > 장기 사용 (B&A) 리뷰 설정 ]

+-----------------------------------------------------------+
| [ 토글 버튼 ] 장기 사용 (B&A) 리뷰 기능 사용하기             |
+-----------------------------------------------------------+

[ 적용 대상 설정 ]
( ) 전 상품 적용
(o) 특정 카테고리/상품 적용
    [ 상품 검색 및 추가 버튼 ] -> (선택된 상품: '시카 진정 크림', '다이어트 쉐이크' 등 5건)

[ 발송 대기 기간 설정 ]
구매 확정(또는 첫 리뷰 작성) 후 알림톡을 발송할 기간을 선택하세요.
[ 셀렉트 박스: ▽ 4주 후 ] (옵션: 2주 / 4주 / 6주 / 8주)

[ 추가 보상(적립금) 설정 ]
일반 리뷰 적립금 외에 B&A 리뷰 작성 시 지급할 추가 적립금입니다.
[ 텍스트 입력: 1,500 ] 원 

[ 텍스트 가이드 설정 ]
B&A 리뷰 작성 페이지에 노출될 가이드 문구를 설정합니다.
[ 텍스트 에어리어:                                            ]
[ 예: "동일한 조명에서 찍은 사진을 올려주시면 적립금 지급 확률 UP!" ]

[ 저장하기 ] 버튼
```

## 2. [프론트엔드] 첫 리뷰 작성 완료 페이지 (Thank You Page)
**설명:** 첫 리뷰 작성을 완료한 유저에게 N주 뒤 알림톡 발송을 예고하여 미리 Before 사진을 찍어두도록 유도하는 화면입니다. PRD의 고민 지점(노출 위치)에서 제안된 '작성 완료 후 노출' 방식을 채택했습니다.

```text
+-----------------------------------------------------------+
|                                                           |
|                       🎉                                  |
|            리뷰 작성이 완료되었습니다!                    |
|             (적립금 500원 지급 완료)                      |
|                                                           |
+-----------------------------------------------------------+

+------------------- [ 하이라이트 배너 ] ---------------------+
|  ✨ 잠깐! 4주 뒤에 한 번 더 혜택을 받으세요!                |
|                                                           |
|  지금 제품을 사용하기 전 사진(Before)을 미리 찍어두세요.    |
|  4주 뒤에 알림톡을 통해 애프터(After) 사진과 함께          |
|  B&A 리뷰를 남겨주시면 [1,500원]을 추가로 드립니다!         |
|                                                           |
|  [ 확인 (미리 찍어둘게요) ]                                |
+-----------------------------------------------------------+

[ 쇼핑 계속하기 ] 버튼
```

## 3. [프론트엔드] 작성 가능한 리뷰 목록 페이지 (마이페이지)
**설명:** 유저가 마이페이지에 들어갔을 때 B&A 리뷰 대상 상품을 명확히 보여주는 화면입니다. PRD 제안에 따라 일반 리뷰와 통합하되 시각적인 뱃지를 주어 구분을 명확히 합니다.

```text
[ 헤더: 작성 가능한 리뷰 (2) ]

+-----------------------------------------------------------+
| [상품 이미지]  시카 진정 크림 50ml                          |
|             옵션: 단품 1개                                 |
|                                                           |
| [ 뱃지: 💰 B&A 전용 리뷰 남기고 1,500원 더 받기! ]          |
|                                                           |
|  [ B&A 리뷰 작성하기 (4주 경과됨) ] 버튼                   |
+-----------------------------------------------------------+

+-----------------------------------------------------------+
| [상품 이미지]  데일리 수분 토너 200ml                       |
|             옵션: 단품 1개                                 |
|                                                           |
|  [ 일반 리뷰 작성하기 ] 버튼                               |
+-----------------------------------------------------------+
```

## 4. [프론트엔드] B&A 전용 리뷰 작성 페이지
**설명:** 실제로 N주 후 접속하여 Before와 After 사진을 업로드하고 내용을 작성하는 폼입니다.

```text
[ 헤더: 장기 사용 (B&A) 리뷰 작성하기 ]
[ 서브텍스트: 시카 진정 크림 50ml ]

+--- [ 가이드라인 박스 ] ------------------------------------+
| 💡 어떻게 찍어야 하나요?                                    |
| 동일한 조명, 동일한 각도에서 찍은 사진을 올려주세요.            |
| (Before 사진은 N주 전 첫 구매 시점의 사진을 권장합니다.)        |
+-----------------------------------------------------------+

[ 1. 사진 등록 (필수 Validation) ]
+-------------------------+   +-------------------------+
|                         |   |                         |
|        📷 Before         |   |         📷 After        |
|    사용 전 사진 등록      |   |    사용 후 사진 등록      |
|                         |   |                         |
+-------------------------+   +-------------------------+
 * 두 장의 사진이 모두 등록되어야 B&A 리뷰로 인정됩니다.

[ 2. 제품은 어떠셨나요? (별점) ]
별점: [ ★ ★ ★ ★ ☆ ]

[ 3. 상세 리뷰 작성 (필수) ]
[ 텍스트 에어리어: 최소 20자 이상 ]
[ 플레이스홀더: "한 달 동안 사용하면서 피부에 어떤 변화가 있었나요? 자세한 후기를 남겨주세요." ]

[ 취소 ]    [ 1,500원 받고 등록하기 (활성화) ] 버튼
```

## 5. [프론트엔드] 상품 상세 페이지 - 리뷰 위젯 노출
**설명:** 다른 구매자들이 B&A 리뷰를 볼 때 노출되는 위젯 UI입니다. 직관적인 비교를 위해 분할 화면 혹은 라벨을 사용합니다.

```text
[ 리뷰 위젯 헤더 - 평점 4.8 / 리뷰 1,234개 ]

[ 탭 메뉴: 전체 리뷰 | 포토/영상 | 👉 B&A 리뷰(New!) ]

+-----------------------------------------------------------+
| [프로필 아이콘] user***  | 평점: ★★★★★ | [ 💰 B&A 리뷰 뱃지 ]|
| 피부타입: 건성 / 고민: 홍조                                 |
|                                                           |
| [ 이미지 영역 (가로 반반 분할 비교) ]                       |
| +-----------------------+-----------------------+         |
| |     [ Before 라벨 ]    |     [ After 라벨 ]     |         |
| |    (사용 전 붉은기)     |     (4주 후 진정됨)     |         |
| +-----------------------+-----------------------+         |
|                                                           |
| 텍스트: "처음 샀을 때 찍어둔 사진이랑 4주 쓰고 난 뒤 사진이랑 |
| 비교해보니 붉은기가 진짜 많이 가라앉았어요. 대박입니다!"      |
|                                                           |
| [ 도움이 됐어요 (12) ]                                     |
+-----------------------------------------------------------+
```
